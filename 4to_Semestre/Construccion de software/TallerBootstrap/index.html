El equipo Omega Squad esta conformado por un grupo interdiciplinario. Conformado por expertos de bases de datos, desarrollo web, APIs, machine learning y cloud.

El equipo se caracteriza por utilizar tecnologías como React, NodeJS, C#, Unity, Unreal Engine, MySQL, PHP, Tensorflow, entre otras.

Para la parte visual se utiliza el lenguaje de diseño Fluent Design creado por Microsoft, utilizando frameworks como MaterialUI y Bootstrap
